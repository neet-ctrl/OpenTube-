El OpenTube no fa servir cap tipus de llibreria de Google o la API de YouTube. El que fa és analitzar el lloc web amb tal d'obtenir l'informació necessària. D'aquesta manera l'aplicació pot utilitzar-se sense tenir els Serveis de Google instal·lats. Tampoc cal un compte de YouTube per fer servir OpenTube. OpenTube és programari lliure, gratuït i de codi obert.
